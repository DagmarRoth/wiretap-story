================================================================================
ADOBE ILLUSTRATOR INSTRUCTIONS
================================================================================

STEP 1: Open any .ai or PDF file in Adobe Illustrator

STEP 2: Click menu → Type → Find/Replace Fonts

STEP 3: In the "Find Font" dialog:
        • Select the missing font from the dropdown (e.g., "Courier-Bold")
        • From "Replace With" dropdown, select "Helvetica-Bold"
        • Click "Replace All"

STEP 4: Repeat for any other missing fonts:
        • Courier-Bold  → Helvetica-Bold
        • Courier       → Helvetica
        • Courier New   → Helvetica
        • Courier New Bold → Helvetica-Bold

STEP 5: After substitution, save the file
        (File → Save) - fonts are now embedded

================================================================================
TROUBLESHOOTING
================================================================================

If fonts still show as missing after substitution:
  1. Check that the substitute font is installed on your system
  2. Try Arial as an alternative to Helvetica
  3. Use Verdana as a last resort

To verify font installation on macOS:
  • Open Font Book application
  • Search for "Helvetica" or "Arial"
  • Ensure the font is enabled (checked)

To verify font installation on Windows:
  • Settings → Apps → Installed apps
  • Search for font name in Control Panel fonts directory
