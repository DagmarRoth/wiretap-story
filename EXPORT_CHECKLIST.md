# AI2HTML Export Checklist — All 3 Charts

## Quick Reference

Export each chart from Illustrator using ai2html script. This creates responsive SVG + PNG versions.

**Output folder**: `ai2html-output/` (keeps exports organized)

---

## Chart 1: Divergence (Cost vs Effectiveness)

**File**: `ai2html-output/chart_01_divergence.ai`

**Steps**:
1. Open `ai2html-output/chart_01_divergence.ai` in Adobe Illustrator
2. File → Scripts → ai2html
3. Settings:
   - ✓ Format: SVG + PNG
   - ✓ Artboards: All
   - ✓ CSS: Inline
4. **Export to same folder** (`ai2html-output/`)
5. Outputs in `ai2html-output/`:
   - `chart_01_divergence.html` (contains full ai2html) ← Extract content from this
   - `chart_01_divergence-Artboard_1.png` (PNG fallback) ← Copy to root

**In index-ai2html.html**: Replace placeholder after line 97 in `#scrolly-divergence`

## After Exporting All 3 Charts

Once ai2html has generated the `.html` and `.png` files in `ai2html-output/`:

**Step 1: Copy PNG files to root**
```bash
# From within wiretap-story/ directory:
cp ai2html-output/chart_01_divergence-Artboard_1.png ./
cp ai2html-output/chart_02_composition-Artboard_1.png ./
cp ai2html-output/sunburst_chart-Artboard_1.png ./
```

**Step 2: Extract and integrate ai2html content**
1. Open `ai2html-output/chart_01_divergence.html` in a text editor
2. Find: `<!-- Generated by ai2html -->`
3. Copy everything from there until the closing `</div>` for the chart box
4. Paste into `index-ai2html.html` where it says `<!-- INSERT chart_01_divergence ai2html HERE -->`
5. Repeat for the other 2 charts

The `.html` files stay in `ai2html-output/` for reference — only the `.png` files and extracted content are used.

## What Each Export Contains

ai2html generates:

```html
<!-- Style section -->
<style media="screen,print">
    #g-chartXX-box { ... }
    #g-chartXX-Artboard_1 { ... }
    .g-aiImg { ... }
    /* + more specific styles */
</style>

<!-- Content section -->
<div id="g-chartXX-box" class="ai2html">
    <div id="g-chartXX-Artboard_1" class="g-artboard" style="width:XXXpx; height:XXXpx;" data-min-width="XXX">
        <img id="g-chartXX-Artboard_1-img" class="g-aiImg" src="chartXX-Artboard_1.png" />
        <svg class="g-aiImg" ...>
            <!-- Full SVG paths and content -->
        </svg>
    </div>
</div>
```

**You need BOTH**: The `<style>` tags AND the full `<div>` structure.

## Integration Steps (Per Chart)

### 1. Open Generated HTML
Look in `chartXX.html` for the ai2html output (starts with `<!-- Generated by ai2html -->`)

### 2. Copy Full Block
```
FROM: <!-- Generated by ai2html -->
TO: </div>  (closing div for g-chartXX-box)
```

Include:
- ✓ `<style>` section
- ✓ Full `<div id="g-chartXX-box">` with all nested content
- ✓ All SVG paths

### 3. Paste into Template
Find the placeholder in `index-ai2html.html`:

**Chart 1**: Search for `<!-- INSERT chart_01_divergence`
**Chart 2**: Search for `<!-- INSERT chart_02_composition`
**Chart 3**: Search for `<!-- INSERT sunburst_chart`

Replace the entire placeholder `<div id="g-chartXX-box">...</div>` with the exported content.

### 4. Verify PNG Files Exist
Check that corresponding PNG files are in the same folder:
- `chart_01_divergence-Artboard_1.png`
- `chart_02_composition-Artboard_1.png`
- `sunburst_chart-Artboard_1.png`
